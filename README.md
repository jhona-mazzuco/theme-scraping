##Web Scraping - PrimeNG Theme

Para utilizar entre na pasta do projeto e faça ``npm install`` e ``node index.js nome-do-tema``.

Exemplo:

```
node index.js mirage-ng 
```


Após isso verifique se seus arquivos estão na pasta ``/tmp``.

---

######Formatos Suportados:
- JSON
- CSS
- SCSS
- SVG
- PNG
- JPG
